# INVARIANTS

## 1. 动作不变量
1. AI 任何时刻都只能从合法动作集合中选择动作
2. 非法动作不得进入状态更新
3. 合法动作生成器是动作唯一入口
4. RAG 不得直接产生可绕过规则系统的执行动作

---

## 2. 牌与手牌不变量
1. 已出牌不能再次出现在任何玩家手牌中
2. 玩家手牌总变化必须与动作一致
3. 手牌减少数量必须等于实际出牌数量
4. 未执行动作前，玩家手牌不得被修改

---

## 3. 状态不变量
1. 任一时刻只能有一个当前行动玩家
2. 玩家轮转必须符合对局规则
3. 一轮结束时桌面约束必须按规则重置
4. 终局后不得继续执行普通出牌动作
5. 状态更新只能由规则引擎完成

---

## 4. 调试不变量
1. 调试模式必须能看到全部 AI 的手牌
2. 调试输出必须包含当前合法动作集合
3. 调试输出必须包含最终选择动作
4. 调试输出必须足以定位规则错误、经验错误、知识错误

---

## 5. RAG 不变量
1. RAG 知识来源只能是掼蛋规则库和人类经验库
2. RAG 结果不能作为合法性最终裁决
3. RAG 不能引入其他纸牌游戏规则
4. RAG 只能作为知识支持，不能替代规则引擎
