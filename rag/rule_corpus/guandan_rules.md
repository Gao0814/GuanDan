# GuanDan Rules (Phase-1)

第一阶段可执行牌型：单张、对子、三张、炸弹（4张同点数）。
pass 是动作类型，不是牌型；仅在跟牌阶段允许。
主动出牌阶段不得选择 pass。
同型比较：单张/对子/三张/炸弹均按点数大小比较。
异型压制：炸弹可以压制单张、对子、三张；炸弹对炸弹按点数比较。
若桌面为炸弹，跟牌只能出更大炸弹或 pass。
若桌面为空，当前玩家必须出非 pass 合法动作。
非法动作不得进入状态更新。
一轮结束条件：领出后其余玩家均连续 pass。
一轮结束后：桌面约束重置，领出权回到最后成功出牌玩家。
终局条件：任一玩家手牌为 0。
终局后不得继续执行普通出牌动作。
