# 贪吃蛇三阶挑战版

这是一个带开始界面、背景音乐、字幕旁白、三阶段规则和怪兽系统的贪吃蛇游戏。

## 运行方法

1. 安装依赖：
   ```
   pip install pygame numpy
   ```

2. 运行游戏：
   ```
   python snake_game.py
   ```

## 操作说明

- 开始界面：按任意键或鼠标左键开始。
- 游戏中：方向键或WASD控制方向。
- 游戏中：按P可暂停/继续。
- 高阶模式：按M切换怪兽惩罚模式（扣分/减节/死亡）。
- 结束界面：按R或鼠标左键重新开始。

## 规则说明

- 初阶：初始蛇身2节。撞墙或自撞直接结束。
- 中阶（达到100分）：速度提升。撞墙-50分-5节，自撞-20分-2节，初始至少5节。
- 高阶（达到220分）：在中阶基础上增加怪兽，怪兽随机移动，碰到怪兽按当前模式惩罚。

## 速度与记录

- 本版已将蛇移动速度整体下调，操作更从容。
- 游戏会自动保存历史最高分到本地文件 `snake_highscore.txt`，并在界面中显示。

## 旁白字幕

- 游戏会根据阶段和事件显示字幕旁白，增加提示与趣味性。

## 注意事项

- 若设备不支持音频，游戏会自动跳过背景音乐，不影响游玩。
